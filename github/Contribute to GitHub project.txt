A couple of suggestions before you start this rewarding journey:
(1) Figure out which technologies you want to use.
(2) Find a project you're interested in on github.com
Use the Explore button in GitHub
(3) Fork the project
(4) Run this project locally
(5) Pick a very small issue you believe you can fix.
(6) Fix it
(7) Generate a Pull request
(8) Pick another small issue. Repeat
